# Number Classifier 1.0

## Description

This project is a Python that analyzes a number and classifies it.
It is designed to determine whether a number is even or odd and positive or negative.

The program is a simple example of how Python functions and conditions work.

## How it works

1. The script defines a number variable.
2. The function checks if the number is even or odd.
3. The program analyzes if the number is positive, negative, or zero.
4. The system returns the classification result.
5. The program prints the result in the terminal.

The application is working correctly and the system is processing the number classification.
